# Google Sheets Booking Format

Create a Google Sheet with these columns:

| Column | Header | Example |
|--------|--------|---------|
| A | Date | 2026-04-05 |
| B | Room | Sea View Suite |
| C | Status | Reserved |
| D | Guest Name | John Doe |
| E | Notes | Check-in 3PM |

---

**Date format:** YYYY-MM-DD (2026-04-05)

**Room options:** "Sea View Suite" or "Garden Sanctuary"

**Status options:** "Reserved" or "Available"

---

**How to share:**
1. File → Share → Publish to web
2. Choose "Comma-separated values (.csv)"
3. Copy the URL and send it to me
